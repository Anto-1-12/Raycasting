# Raycasting
Ce projet a pour but de comprendre et reproduir le Raycasting, similaire a celui de Wolfenstein 3D ou Doom
# Pourquoi autant de version?

Chaque version de raycasting est différent, la premiere version étant raycasting.py
Les version 3 et 4 sont très similaires mais differentes car la version 4 est totalement recodé de zéro pour éliminer tout les bugs causé par le développement mal ordoné
La version 5 est une tentative de recréer un moteur de Raycasting avec ma propre méthode mais beaucoup de bug de vision et surtout des bugs génant visuellement sont présent, il est quand même bien mieux optimisé
J'ai codé la version 6 après avoir apris et compris le fonctionnement des moteurs 3D, il est donc plus complèt, il y a quand même des bugs visuelles qui devraient pas être très difficile a enlever
